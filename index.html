My developer origins stems from a past desire to improve daily tasks in most of my past professions. I realized I could improve the end-user experience while implementing certain ideas for greater workplace efficiency. Eventually, I began developing full web applications.

What I Do

Below are some projects I have worked on as a developer. For some projects, I was the sole developer, and in others I had a particular role. Rails/React

NightStock

A web application utilizing an external CDN to provide a media sharing platform for photography taken during the night. Tech used: React.js, Redux, JavaScript (ES6), CSS, SCSS, HTML5, jQuery, a PostGreSQL database, Ruby on Rails, and RESTful API requests.

Django/React

CarsForMe

A web application providing vehicle purchasing information with a custom API for users. Tech used: Django, Python, RESTful requests, Google Maps API, Google Places API, Google Search API, Edmunds API, CarQuery API, React.js, Redux, Infinite Scroll.

A flash game made purely to showcase the potential of JavaScript (ES6) itself. Tech used: Canvas.js, CSS, HTML5, jQuery, and JavaScript. The game focuses on using event listeners, and various keybindings to provide players infinite gameplay.
